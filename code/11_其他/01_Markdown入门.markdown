# 2.字体
*斜体* 快捷键 Ctrl + B 加粗
**粗体** 快捷键 Ctrl + I 斜体
***粗斜体***

# 3.换行
1. 直接在一句话后敲两下空格  
2. 两句话中间加一个空行
3. 编辑中像让一行文字在显示的时候换行, 插入<br/>即可

# 4.引用
> 引用
>> 引用的引用
>>> 应用的引用的引用

# 5.链接
[链接名称] (链接地址)
<链接地址>

# 6.图片

# 7.列表

2. 有序列表 数字加上.再加一个空格
